# Notice

This repository is a public educational release of Team 2544X Bolted's robot-code architecture.

The project is buildable and uploadable, but the complete working recorded-auton replay implementation has been replaced with a guide-only module. The goal is to explain the engineering concepts without turning our robot-specific field tuning into a copy-ready template.

Unless a separate `LICENSE` file is added by the team, this repository is not published as an open-source starter kit. You may read it, learn from it, and credit the concepts. Do not rebrand it and submit it as another team's original software.
