# Legacy Archive

This folder keeps older 2544X Bolted files public for reference.

## Folders

- `push_back/`: older VEXcode `.v5python` files from the Push Back strategy era.
- `replay_archive/`: old replay archive file.
- `web/`: old repository web/documentation page.

These files are not part of the current PROS build. They are kept open because they show team history, old experiments, and earlier design direction.

## Current Code

Use the root `src/`, `include/`, `firmware/`, and `project.pros` files for the current buildable PROS project.
